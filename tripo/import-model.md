Import Model
Import private model
POST https://api.tripo3d.ai/v2/openapi/task
Request
type: Must be set to import_model.
file: Specifies the file input.
object (Strongly Recommended): The information you get from upload (STS), please read Docs/Upload (STS).
bucket: Normally should be tripo-data.
key: The resource_uri got from session token.
Response
task_id: The identifier for the successfully submitted task
Behaviour
Once the task moves out of the waiting queue, it typically completes within a few seconds. You will be able to use post-process on imported model as same as the model generated by us.

Example
Request:

Library:
curl
export APIKEY="tsk_***"
curl https://api.tripo3d.ai/v2/openapi/task \
-H 'Content-Type: application/json' \
-H "Authorization: Bearer ${APIKEY}" \
-d '{
	"type": "import_model",
	"file": {
	  "object":{
	    "bucket":"tripo-data",
	    "key":"***"
	  }
	}
}'
unset APIKEY
Response:

  }
